`难度：`简单

`题目：`给你两个字符串 word1 和 word2 。请你从 word1 开始，通过交替添加字母来合
并字符串。如果一个字符串比另一个字符串长，就将多出来的字母追加到合并后字符串的末
尾。返回 合并后的字符串 。

`思路：`

- 我们通过双指针的方法，来交替拼接字符串。

`答案：`

```js
/**
 * @param {string} word1
 * @param {string} word2
 * @return {string}
 */
var mergeAlternately = function (word1, word2) {
  let res = '';
  let i = 0;
  let j = 0;
  while (i < word1.length || j < word2.length) {
    if (i < word1.length) {
      res += word1[i];
      i++;
    }
    if (j < word2.length) {
      res += word2[j];
      j++;
    }
  }
  return res;
};
```
