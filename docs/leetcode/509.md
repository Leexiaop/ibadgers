`难度：`简单

`题目：`斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0
和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

```
F(0) = 0,F(1) = 1
F(n) = F(n-1) + F(n-2),其中n>1
```

给定 n，计算 F(n)

`思路：`递归

- 题目中，以及描述的很清楚了，我就按照题目中的描述写代码即可

`答案：`

```js
/**
 * @param {number} n
 * @return {number}
 */
var fib = function (n) {
  if (n === 0) return 0;
  if (n === 1) return 1;
  return fib(n - 1) + fib(n - 2);
};
```
